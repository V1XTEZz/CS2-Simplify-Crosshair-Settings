# CS2-Simplify-Crosshair-Settings
This CFG will help you to customize ur crosshair in CS2.

<p>Download the config and put it: <blockquote>...\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg</blockquote>
<br>Next U can bind it or copy the [Alias SCS] to ur config and after just write in console SCS.</p>

<br><b>The Settings</b>:
<blockquote><p><i>CGap</i> - <code>toggle between (-2, -3, -4, -5)</code></p>
<p><i>CSize</i> - <code>toggle between (1, 2, 2.5, 3)</code></p>
<p><i>CThickness</i>/<i>CThicc</i> - <code>toggle between (0.5, 1)</code></p>
<p><i>CStyle</i> - <code>toggle between (2, 4, 5)</code> <i>(2 - Absolute Dynamic, 4 - Static, 5 - Shots Only Dynamic)</i></p>
<p><i>CAlpha</i> - <code>toggle between (120, 150, 180, 200, 220, 250)</code></p>
<p><i>CDot</i> - <code>toggles dot</code></p>
<p><i>CT</i> - <code>toggles T style of crosshair</code></p>
<p><i>Outline</i> - <code>toggles outline</code></p>
<p><i>CFF</i> - <code>toggles Frendly warning</code></p>
<p><i>CGapUWV</i> - <code>toggles use weapon value</code></p>
<p><i>CCol[num]</i> - <code>toggles color (1-5)</code></p>

<br><i>SCSize</i> - <code>toggles sniper crosshair (1, 2)</code>
</blockquote>
